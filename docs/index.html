Each sample shows one episode across all correction attempts. Attempt 1 is the vanilla VLA policy with no correction. Subsequent attempts apply VLM-generated action-bias corrections that evolve based on failure analysis. The robot learns to succeed without any weight updates — only prompt-level in-context correction.
